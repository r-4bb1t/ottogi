# OTTOGI STREAMLIT
## SET UP VENV
### Windows
1. set up venv
```shell
python -m venv venv
```
2. venv activate
```shell
venv\Scripts\activate
```
3. install dependencies
```shell
pip install -r requirements.txt
```

### Ubuntu
1. set up venv
```shell
python3 -m venv venv
```
2. venv activate
```shell
source venv/bin/activate
```
3. install dependencies
```shell
pip3 install -r requirements.txt
```

## RUN
```shell
streamlit run Home.py
```
